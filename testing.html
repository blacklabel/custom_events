Custom Events

by Black Label

Elements with custom events:

  • Chart Title - Main chart title
  • Subtitle - Chart subtitle
  • Axis Titles - X and Y axis titles
  • Axis Labels - Tick labels on axes
  • Crosshair - Vertical line on X axis
  • Plot Line Labels - Labels on plot lines
  • Plot Band Labels - Labels on plot bands
  • Data Labels - Values displayed on series points

How to test:

  • Click on any chart element from the list
  • Watch the event log for real-time feedback
