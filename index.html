Installation

NPM npm install highcharts-custom-events
CDN <script src="https://unpkg.com/highcharts-custom-events@latest/dist/customEvents.js"></script>

Usage example

// After loading Highcharts and the plugin
Highcharts.chart('container', {
    title: {
        text: 'Chart Title',
        events: {
            click: function() {
                console.log('Title clicked!');
            },
            dblclick: function() {
                console.log('Title double-clicked!');
            }
        }
    }
});
